# CELSON 网站远程操作指南

> 适用：手机 / 平板 / 任何电脑  
> 改完自动上线，无需登录 Cloudflare

---

## 一、改价格（最常用）

### 📱 手机操作

1. 打开浏览器，访问：  
   `https://github.com/XavierHeng/celson-website`

2. 点击 **js** 文件夹 → 点击 **prices.js**

3. 点击右上角 **✏️ 铅笔图标**（Edit this file）

4. 修改数字，例如：
   ```
   'gyp-std-12': 12.75,   // 改成你要的价格
   'gyp-mr-12': 18.00,
   ```

5. 改完后，点击右上角绿色 **「Commit changes」** 按钮

6. 在弹出的框里写一句说明（如 "更新石膏板价格"）→ 再点 **「Commit changes」**

7. **等 60 秒**，刷新 `celson.ltd` 就能看到新价格 ✅

---

### 💻 电脑操作

1. 浏览器打开 `https://github.com/XavierHeng/celson-website`

2. 点进 **js/prices.js** → 点 **✏️ 编辑**

3. 修改价格数字 → 点 **Commit changes**

4. 等待 1 分钟，网站自动更新

## 四、替换图片

**方法：在 GitHub 网页上操作**

1. 打开 `https://github.com/XavierHeng/celson-website`

2. 找到要替换的图片位置：
   - 产品图在 `assets/` 文件夹
   - Shop 缩略图在 `assets/shop/` 文件夹

3. 点击目标图片 → 点右上角 **🗑️ 删除图标** → Commit

4. 回到 `assets/` 文件夹 → 点 **「Add file」** → **「Upload files」**

5. 拖入新图片 → 文件名要和原来一样 → Commit

6. 等待 2 分钟上线

## 五、常见问题

**Q: 改了没生效？**  
A: 等 1-2 分钟后刷新浏览器（按 Ctrl+F5 强制刷新）。如果还不生效，在 GitHub 点 **「Actions」** 标签查看部署状态。

**Q: 改错了怎么恢复？**  
A: GitHub 有历史记录。点文件 → 右上角 **「History」** → 找到之前的版本 → **「Revert」**

**Q: 忘记账号密码？**  
A: GitHub 账号：`XavierHeng`。密码忘记去 `github.com` 点 "Forgot password"。

**Q: 能把账号分享给员工吗？**  
A: 不要直接给密码！去 GitHub 仓库 Settings → Collaborators → 添加员工 GitHub 账号。对方只能用自己账号操作，可随时移除。
