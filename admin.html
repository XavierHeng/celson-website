⚙️ 连接 GitHub

需要一个 GitHub Personal Access Token 才能自动保存和发布。

🔐 重要：GitHub 账号密码由管理员保管
创建 Token 需要使用 XavierHeng 的 GitHub 账号登录。
如需账号密码，请联系：Xavier · 📱 13302802833

1. 打开 github.com/settings/tokens
2. 如未登录，先用 Xavier 提供的账号密码登录 GitHub
3. 点 "Generate new token" → 选 Fine-grained token（或 Classic token → repo 全选）
4. Repository access: Only select repositories → 选 XavierHeng/celson-website
5. Permissions → Contents: Read and write
6. 点 Generate → 复制 Token 粘贴到下面
