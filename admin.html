⚙️ 连接 GitHub

需要一个 GitHub Personal Access Token 才能自动保存。

1. 打开 github.com/settings/tokens
2. 点 "Generate new token" → 选 Fine-grained token
3. Repository access: Only select repositories → 选 XavierHeng/celson-website
4. Permissions → Contents: Read and write
5. 点 Generate → 复制 Token 粘贴到下面
