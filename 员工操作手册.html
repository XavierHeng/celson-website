② 在浏览器新标签页打开
https://github.com/settings/tokens?type=beta		 GitHub 令牌生成页面
③ 点 Generate new token（经典版） 如果用新版菜单，点 Fine-grained tokens → Generate new token
④ 填写：Note = celson-admin，Expiration = No expiration 永不过期，不用重复操作
⑤ 勾选权限：repo（全部勾上即可） 允许代码读写
⑥ 点底部 Generate token
⑦ 复制生成的 token（格式 github_pat_xxxx...） ⚠️ 离开页面后无法再查看！
⑧ 回到后台 → GitHub设置面板 → 粘贴 → 保存 之后自动记住，无需再次设置
⚠️ Token 不要发给任何人！ 它相当于你的 GitHub 密码。如果 Token 丢失或泄露，回到 GitHub 页面删掉旧的重新生成即可。

2 修改产品价格

2.1 单个产品改价
步骤操作说明
① 登录后台，进入产品管理主界面 所有产品以卡片形式排列
② 找到要改价的产品卡片 可按分类筛选加速查找
③ 在 ¥ 输入框中输入新价格 修改后输入框变黄色，表示已改动但未发布
④ 可继续修改多个产品价格 所有改动卡片都会标黄
⑤ 点底部绿色 「📤 发布到网站」 确认改动的产品列表，点确认
⑥ 等待约 60 秒 → 提示「✅ 发布成功」 可点「查看部署状态」链接确认
⑦ 打开官网确认 访问 https://celson.ltd/shop.html，硬刷新（Ctrl+F5）
2.2 批量调价 NEW
🚀 适合场景：全线涨价/降价、促销活动统一调价、某个分类集体调价。
步骤操作说明
① 点击产品的 ☑ 勾选框（多选） 可使用顶部「☑ 全选」快速选中所有产品
② 在顶部批量操作栏中，选择模式：百分比 或 固定金额 百分比：如 10% = 所有选中产品涨价10%
③ 输入数值（正数=涨价，负数=降价） 例如输入 5 → 全部涨5%
④ 点 「应用」按钮 所选产品价格立即更新，输入框变黄
⑤ 确认无误后点 「📤 发布到网站」 与单改同样的发布流程
⚠️ 改完后必须点「发布」才算生效！ 只改数字不点发布，网站价格不会变。黄色=未发布，白色=已发布。

3 产品增删改 NEW

3.1 新增产品
✨ 添加新产品会自动归入你选择的分类，前台 Shop 页面即时显示，无需改代码。
步骤操作说明
① 点顶部 「➕ 新增产品」 按钮 弹出新增产品表单弹窗
② 填写 SKU 编号 格式：分类缩写-规格，如 gyp-fr-12；必须是唯一的
③ 填写产品名称（英文） 显示在网站上的标题，如 Fire-Rated Gypsum Board 12mm
④ 选择所属分类 从下拉框选已有分类（如无合适分类，先在分类管理新建）
⑤ 填写规格、备注、单位、起订量 规格：产品尺寸和特性；备注：发货地/数量说明；单位：如 /m² EXW
⑥ 设置价格（¥） 人民币 EXW 价格
⑦ 上传产品图片（可选） 支持 JPG/PNG/WebP，推荐 500×500 px 正方形
⑧ 点 「新增」 确认 → 然后点 「📤 发布到网站」 产品卡片出现 → 发布后前台生效
3.2 编辑产品名称 / 规格
操作方法
改产品名称 双击产品卡片上的名称文字 → 编辑 → 按 Enter 确认（Esc 取消）
改规格描述 双击规格文字 → 编辑 → 按 Enter 确认
改备注（发货/数量） 双击备注文字 → 编辑 → 按 Enter 确认
改单位 双击单位文字（如 /m² EXW）→ 编辑 → 按 Enter 确认
💡 双击编辑后字段变黄色，表示已改动。所有改动一起发布就行。
3.3 删除 / 隐藏产品
操作效果如何操作
隐藏 后台仍可见（灰色卡片），前台不显示 点产品卡片上的「👁 隐藏」按钮
显示 取消隐藏，前台恢复显示 勾选底部「显示已隐藏」→ 点「显示」按钮
删除 彻底移除产品，数据不可恢复 点产品卡片上的「🗑 删除」→ 确认
⚠️ 建议先用「隐藏」而非「删除」。 隐藏的产品数据保留，以后可以恢复。删除后无法找回。

4 替换产品图片

步骤操作说明
① 登录后台，找到要换图的产品 每个产品卡片左侧有产品图片区域
② 鼠标悬停在图片上（手机：点击图片） 出现半透明提示「📷 换图」
③ 点击 「📷 换图」 弹出文件选择窗口
④ 选择本地图片文件 支持 JPG / PNG / WebP，推荐 500×500 px 正方形，≤2MB
⑤ 确认预览正确 图片区自动显示新图预览，卡片标黄
⑥ 点 「📤 发布到网站」 与改价相同的发布流程
💡 可以同时改价格 + 换图片 + 改名称 → 一次性发布，不需要分多次操作。

5 分类与排序管理 NEW

5.1 分类管理
操作方法说明
新增分类 点 「📂 分类管理」 → 输入名称 → 点「+ 添加」 新增后可在新增产品时选择该分类
改名分类 分类管理弹窗 → 点右侧「✏️」→ 输入新名称 → 回车 所有该分类下的产品自动更新
删除分类 分类管理弹窗 → 点「🗑」 ⚠️ 如果已有产品归属该分类，不可删除，需先转移产品
5.2 分类筛选
操作方法
按分类筛选 点顶部分类胶囊（如「Gypsum Boards」）→ 只显示该分类产品
显示全部 点「全部」胶囊 或 清除筛选
5.3 产品排序
操作方法说明
上移一个 点产品卡片右下角 ▲ 按钮 产品在列表中向前移动一位
下移一个 点产品卡片右下角 ▼ 按钮 产品在列表中向后移动一位
发布 排序后点 「📤 发布到网站」 前台产品展示顺序同步更新
💡 排序主要用于控制 Shop 页面前台展示顺序。排前面的产品在前台更靠前。

6 常见问题

问题原因解决方法
改了价格但网站没变？ 没点发布 回到后台，点「📤 发布到网站」，等60秒刷新
发布后价格还是旧的？ 浏览器缓存 按 Ctrl+F5（电脑）或清除浏览器缓存（手机）刷新
登录提示密码错误？ 输入有误 用户名 Celsonbetty，密码 86880707，注意大小写
发布按钮灰色/无法点击？ 没有任何改动，或未连接 GitHub 先修改价格/名称/图片，按钮变绿后即可点击。如仍灰色，检查「⚙️ GitHub设置」是否已保存 Token
新增产品后前台看不到？ 没发布 或产品被隐藏 检查卡片是否标「已隐藏」→ 解除隐藏 → 发布
手机上看不清卡片？ 屏幕太小 横屏旋转手机，或使用电脑。推荐电脑操作新增/删除，手机操作改价
改错了怎么恢复？ 发布前：点底部 「↩ 撤销全部改动」，一键回滚
发布后：立即联系管理员在 GitHub 回滚
批量调价结果不对？ 勾选了不应改的产品 点「↩ 撤销全部改动」还原，重新选择后再应用
图片上传后边缘被裁切？ 非正方形图片 后台会自动裁剪为正方形居中。建议先用手机相册裁剪为 1:1 比例
多个员工同时改了价格？ 多人操作冲突 后发布的人会覆盖先发布的内容。建议操作前先沟通，或操作前刷新页面拉最新数据

7 重要提醒

事项说明
🔐 密码保密 登录密码和 GitHub Token 仅限内部员工使用，勿外传
📤 一次发布 = 全部上线 发布按钮会把所有未发布的改动（改价+改名+换图+排序+新增）一次性提交。多人操作时请注意沟通协调
🟡 黄色 = 未发布 任何字段变黄（价格/名称/规格/图片）都表示已改动但尚未发布
📱 手机也能操作 出差在外、不在公司也能用手机改价，无需安装任何 APP
⏱ 生效延迟 ~60 秒 从点发布到网站显示新内容大约需要 1 分钟（GitHub Actions 构建 + Cloudflare 全球 CDN 同步）
🖼 图片规范 建议使用 500×500 px 以上正方形图片，最大不超过 2MB，格式推荐 WebP 或 PNG
🗑 删前隐藏 不确定要不要删除的产品，先用「隐藏」代替「删除」。隐藏=保留数据，删除=不可恢复
🔄 操作前先刷新 多人操作时，打开后台后先刷新页面，确认看到的都是最新数据，避免覆盖他人改动
☎ 技术支持 如遇系统故障，联系管理员或 IT 支持
